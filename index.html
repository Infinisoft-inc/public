Brainstack Framework

A powerful framework for building scalable and modular applications with seamless state management, real-time communication, and cognitive memory capabilities.

Modular Architecture

Build applications using independent, reusable micro packages that can be combined to create complex systems.

Cognitive Memory System

Implement human-like memory patterns with attention, short-term, and long-term memory layers for intelligent data management.

Real-time Communication

Enable seamless WebSocket communication between components with built-in pub/sub capabilities.

State Management

Manage application state effortlessly with a powerful state management system designed for complex UI components.

Authentication & Security

Integrate robust authentication with support for multiple providers including AWS Cognito and Keycloak.

Developer Tools

Boost productivity with built-in logging, debugging, and development tools.

Core Packages

@brainstack/core

The main package bundling essential micro packages for quick start.

@brainstack/memory

Cognitive memory system for intelligent data management.

@brainstack/inject

Lightweight dependency injection for better code organization.

@brainstack/hub

Event-driven communication hub for real-time features.

Getting Started

Install the core package to get started:

npm install @brainstack/core

Import and use the components you need:

import { createLogger, createHub, createStateManager } from '@brainstack/core';

// Create instances
const logger = createLogger();
const hub = createHub();
const stateManager = createStateManager();
