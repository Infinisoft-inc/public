What is Micro Hub?

Micro Hub is a TypeScript library that allows users to easily subscribe to and emit events with a variety of features to make event handling simple and efficient. It supports both string and regular expression event names, giving users full control over their event handling. Additionally, Micro Hub prevents event duplication, ensuring that events are handled efficiently and effectively.
